ðŸ”§ How It Works

The auto-styling is implemented in the library's entry point: 
// src/index.ts
import './lib/auto-styles'  // This line triggers style injection

export * from './components/atoms'
export * from './components/molecules'
// ... other exports

The auto-styles.ts file contains the CSS and automatically injects it into the document head when the library is imported.

ðŸ“± Mobile Optimizations

Additional mobile-friendly styles are also applied:
  • -webkit-tap-highlight-color: transparent - Removes the blue highlight on mobile taps
  • html, body { height: 100% } - Ensures full height for mobile layouts
  • box-sizing: border-box - Makes sizing more predictable across devices
